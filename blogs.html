Question-1: What are the different ways to select an element in the DOM?

getElementById() id diye select kora jai. (Element) ata ar return type.
getElementsByClassName() class diye select kora jai. (HTMLCollection) ata ar return type.
getElementsByTagName() tag nam diye select kora jai. (HTMLCollection) ata ar return type.
querySelector() css sector ar madthome select kora jai. (Element) ata ar return type.
querySelectorAll() eker othik element select kora jai. (NodeList) ata ar return type.

Question-4: What is event bubbling in the DOM?

Event Bubbling holo jokhon ekti child element a event trigger hoi, tokhon sei event parent element porjonto upore uthe jai. Eti bottom-to-top (child theje parent) gromon kore.
